# Inventory & Stock System

The Inventory system manages stock tracking, warehouse logistics, and transaction logs:

- **Inventory Categories (`inventory_categories`)**: Classifies items (stationary, classroom guides, cleaning consumables, kitchen supplies).
- **Warehouses (`warehouses`)**: Physical storage locations where materials are kept.
- **Inventory Items (`inventory_items`)**: Tracks actual stock volumes, units, and minimum restock limits.
- **Inventory Transactions (`inventory_transactions`)**: Chronologically records movements (`purchase`, `usage`, `transfer`, `adjustment`), keeping item quantities accurate.
