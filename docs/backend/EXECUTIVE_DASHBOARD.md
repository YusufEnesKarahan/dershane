# Executive Dashboard Module

The Executive Dashboard provides system administrators with an aggregated view of key business and academic indicators:

- **Metrics Included**: Student/Teacher ratios, branches count, today's lessons, marked attendance counts, unexcused absence rates, exam averages (TYT/AYT), financial collections, outstanding debt, homework rates, and communication notifications.
- **Snapshot Logic (`dashboard_snapshots`)**: Persists computed KPI arrays for historical snapshot reporting.
