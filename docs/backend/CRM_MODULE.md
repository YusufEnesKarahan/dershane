# CRM & Lead Management Module

The CRM module enables track pre-enrollment sales operations for candidate students:

- **Candidate Lead (`crm_leads`)**: Tracks candidates contact info, WhatsApp, desired preparation program (YKS/LGS), source, and status.
- **Activity Timber (`lead_activities`)**: Logs timeline updates (such as SMS/Email sent, calls, offer).
- **Advisors Assignments (`lead_assignments`)**: Logs advisor switches and branch assignments.
