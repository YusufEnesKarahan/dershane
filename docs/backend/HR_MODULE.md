# Human Resources (HR) Module

The HR Module manages the organization workforce registry, department groupings, and job roles/positions:

- **Departments (`departments`)**: Work divisions with Manager associations.
- **Positions (`positions`)**: Roles within a department, level indicators (Junior, Mid, Senior, Lead), and baseline salaries.
- **Employees (`employees`)**: Core workforce records including user association, personal identifiers, contact details, contract types, and ibans.
