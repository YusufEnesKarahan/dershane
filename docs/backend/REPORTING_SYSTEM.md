# Reporting & Scheduled Reports Module

The Reporting engine enables exporting system data in multiple formats and scheduling automated reports:

- **Formats**: Supports PDF, Excel, and CSV format exports tracked via `report_exports`.
- **Scheduled Reports (`report_schedules`)**: Registers Cron expressions and email recipients to automate report generation in the background.
- **Executive Reports (`executive_reports`)**: Registers custom written analytical assessments by administrators.
