# Dashboard Widgets

The framework provides ready-to-use aesthetic widgets under `resources/views/components/admin/widgets`.

## Stat Widget
`<x-admin.widgets.stat title="Total Students" value="1,240" trend="12" />`
Displays a primary metric with an optional percentage trend indicator (green for positive, red for negative).

## Chart Placeholder
`<x-admin.widgets.chart-placeholder title="Revenue Growth" />`
A structured container representing the space and UI for future Chart.js or ApexCharts implementations.
