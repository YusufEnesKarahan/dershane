# Dershane / Etüt Merkezi / Eğitim Kurumu Yönetim Sistemi

Profesyonel ve production seviyesinde geliştirilmiş, tek kod tabanı (single codebase) üzerinden 3 farklı paket olarak dağıtılabilen modern bir Eğitim Kurumu ERP/SaaS platformu.

## 🚀 Teknolojik Altyapı
- **Framework:** Laravel 12 (PHP 8.4)
- **Frontend:** Blade, Vite, TailwindCSS, AlpineJS
- **Veritabanı:** SQLite (Geliştirme), MySQL (Canlı)
- **Standartlar:** PSR-12, Strict Types, PHPStan, Laravel Pint

## 📦 Paket Yapısı
Platform, dinamik Feature Flag mekanizması ile tek bir kod tabanı üzerinden aşağıdaki 3 sürümü destekler:
1. **Version 1 (Kurumsal Web Sitesi):** Tanıtım, programlar, iletişim ve temel kayıt/başvuru altyapısı.
2. **Version 2 (Kurumsal + Yönetim):** Version 1 özelliklerine ek olarak temel kurum yönetim paneli ve yönetim işlevleri.
3. **Version 3 (Full Eğitim ERP):** Gelişmiş dershane, etüt, rehberlik, finans, sınav, öğrenci/veli/öğretmen portalları içeren eksiksiz ERP çözümü.
